### 🎯 예제 8: 여러 데이터 검색 - %in% 연산자 💡

#### **😊 SQL과 R의 비교 연산자**

| SQL | R | 설명 |
|-----|---|------|
| `IN` | `%in%` | 여러 값 중 하나와 일치 |
| `LIKE` | `grep` | 패턴 매칭 |
| `IS NULL` | `is.na` | NULL 값 확인 |
| `BETWEEN A AND B` | `(emp$sal >= A) & (emp$sal <= B)` | 범위 검색 |
